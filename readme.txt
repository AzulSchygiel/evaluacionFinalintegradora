───── Evaluación Final Integradora ─────

Consigna:

✓ Diseñar una aplicación web que contenga 4 secciones, una de las cuales será la página inicial de nuestro sitio, las restantes deben estar relacionadas con el tema escogido.
✓ La aplicación debe conectarse a base de datos, mediante Mysqli o PDO.
✓ Para el desarrollo se podrá utilizar PHP nativo orientado a objetos, empleando MVC o el frameworks Laravel.
✓ Incluir un archivo readme.md a modo que explicite la lógica y las herramientas utilizadas.
✓ Incluir un video a modo de documentación, que dé cuenta todas las funcionalidades de la aplicación, incluyendo las operaciones en la base de datos.
✓ Incluir URL del host online.

Consideraciones generales:

⋆ Se deben entregar en una carpeta todos los archivos necesarios para el funcionamiento de la aplicación.
⋆ En el caso de utilizar Laravel, incluir todos los archivos la instalación de con las modificaciones a los archivos de configuración para que todo funcione correctamente y 
  los archivos correspondientes a los modelos, controladores y vistas ubicados en las carpetas destinadas a tal fin.
⋆ En ambos casos incluir dump SQL: tiene que ser denominado de la siguiente base de datos: apellido_nombre.sql. Es imprescindible para la evaluación.
⋆ Permitir acceso por usuario:admin y password:admin.
⋆ La entrega puede realizarse en zip o repositorio github o similar y se debe incluir la  URL online.

-------INFORMACIÓN DEL TRABAJO---------

Las vistas que tiene el sitio web son:
1- Inicio de sesion
2- Registro
3- Home (Solo accede el admin)
   |_ Editar registro de la Base de datos
   |_ Eliminar registro
4- Perfil
5- Salir
+ Error

La aplicación se conecta a la Base de datos con Mysqli.

Acceso a la vista hacia la Base de datos desde el home con nombre de usuario:admin, email: amin y password:admin.

En el desarrollo se utilizó PHP nativo orientado a objetos, JavaScript, LocalStorage, y estructura MVC.
